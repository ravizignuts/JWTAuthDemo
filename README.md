# Implement JWT Authentication
# Install Project
Clone this repository
-  composer install
-  cp .env.example .env
-  php artisan migrate
-  php artisan key:generate
-  php artisan serve
 
 ## Pre-Required Packages
- laravel/framework : 10.0
- php               : 8.1
- tymon/jwt-auth    : 2.0
